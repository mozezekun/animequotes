# AnimeQuotes

A modern, full-stack web application for browsing, searching, and sharing inspirational anime quotes. Built with React, TypeScript, Express, and ArangoDB.

ğŸŒ **Live Site**: [https://animequotes.sbs](https://animequotes.sbs)

## Features

- **Browse Quotes**: View a curated collection of anime quotes from popular series
- **Search**: Find quotes by text, character, anime title, or tags
- **Like System**: Like your favorite quotes
- **Add Quotes**: Contribute new quotes to the collection
- **Random Quote**: Get a random inspirational quote
- **Responsive Design**: Beautiful UI built with Tailwind CSS and shadcn/ui components
- **SEO Optimized**: Full meta tags, sitemap, and social media integration

## Tech Stack

### Frontend
- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **TanStack Query** - Server state management
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI component library
- **Framer Motion** - Animations
- **React Three Fiber** - 3D graphics (optional)

### Backend
- **Express 5** - Web server
- **ArangoDB** - NoSQL database
- **Zod** - Schema validation
- **CORS** - Cross-origin resource sharing
- **dotenv** - Environment configuration

## Getting Started

### Prerequisites

- Node.js 22+ (LTS recommended)
- pnpm 10+ (package manager)
- ArangoDB instance (or use the configured cloud instance)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/animequotes.git
cd animequotes
```

2. Install dependencies:
```bash
pnpm install
```

3. Create a `.env` file in the root directory:
```env
# Optional: Custom ping message
PING_MESSAGE=pong

# Database configuration (if using your own instance)
# ARANGO_URL=your_arango_url
# ARANGO_DB=your_database_name
# ARANGO_USERNAME=your_username
# ARANGO_PASSWORD=your_password
```

4. Start the development server:
```bash
pnpm dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `pnpm dev` - Start development server with hot reload
- `pnpm build` - Build for production (client + server)
- `pnpm build:client` - Build only the client
- `pnpm build:server` - Build only the server
- `pnpm start` - Start production server
- `pnpm test` - Run tests
- `pnpm typecheck` - Run TypeScript type checking
- `pnpm format.fix` - Format code with Prettier

## API Endpoints

### Quotes

- `GET /api/quotes` - Get all quotes with pagination and search
  - Query params: `limit`, `offset`, `search`
- `GET /api/quotes/random` - Get a random quote
- `POST /api/quotes` - Add a new quote
  - Body: `{ quote, character, anime, tag? }`
- `POST /api/quotes/like` - Like a quote
  - Body: `{ quoteId }`

### Other

- `GET /api/ping` - Health check endpoint
- `GET /sitemap.xml` - SEO sitemap

## Deployment

### Docker

Build and run with Docker:

```bash
docker build -t animequotes .
docker run -p 8080:8080 animequotes
```

### Netlify

This project is configured for Netlify deployment with serverless functions.

1. Connect your repository to Netlify
2. Configure build settings:
   - Build command: `pnpm build`
   - Publish directory: `dist/spa`
3. Add environment variables in Netlify dashboard
4. Deploy!

## Database Schema

### Quote Document

```typescript
{
  _id?: string;           // Auto-generated by ArangoDB
  quote: string;          // The quote text
  character: string;      // Character name
  anime: string;          // Anime title
  tag?: string;           // Category/tag
  likes: number;          // Like count
  createdAt?: string;     // ISO timestamp
}
```
